# OK MOBILE

A modern Single Page Application for smartphone sales, repairs, and accessories.

## Features

- Responsive Design
- Firebase Authentication
- Firestore Database
- Firebase Storage
- Dynamic Products
- Dynamic Hero Banner
- Shop Gallery
- Theme Customization
- Social Links Management
- Admin Dashboard
- GitHub Pages Compatible

---

# Firebase Setup

## 1. Create Firebase Project

Open:

https://console.firebase.google.com

Click

Create Project

Example name

OK MOBILE

---

## 2. Enable Authentication

Authentication

Sign In Method

Enable

Email / Password

---

## 3. Create Firestore Database

Mode:

Production

---

## 4. Enable Storage

Create Storage

Choose your region

---

## 5. Get Firebase Config

Project Settings

General

Scroll down

Add Web App

Copy:

const firebaseConfig = {

apiKey:"",

authDomain:"",

projectId:"",

storageBucket:"",

messagingSenderId:"",

appId:""

};

Paste it inside

js/firebase.js
